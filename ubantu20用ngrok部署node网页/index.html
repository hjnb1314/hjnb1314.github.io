现象: Sequelize 无法连接到 MySQL，出现错误 Access denied for user ‘root‘@’localhost’。

解决方案:

确认 MySQL 用户名和密码正确，并且有足够权限。
修改 server.js 中的 Sequelize 配置，确保数据库名、用户名、密码和主机名正确。
